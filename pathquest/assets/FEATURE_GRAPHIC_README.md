# Feature Graphic (1024x500px) for Google Play

## Files Created

1. **`feature-graphic.svg`** - SVG version (scalable, can export to PNG)
2. **`feature-graphic.html`** - HTML version (open in browser, screenshot)

## How to Convert to PNG

### Option 1: SVG → PNG (Recommended)
1. Open `feature-graphic.svg` in:
   - **Inkscape** (free): File → Export PNG → Set to 1024x500px
   - **Adobe Illustrator**: File → Export → PNG → 1024x500px
   - **Figma**: Import SVG → Export as PNG → 1024x500px
   - **Online**: Use https://cloudconvert.com/svg-to-png

### Option 2: HTML → PNG (Easiest)
1. Open `feature-graphic.html` in Chrome/Edge
2. Right-click → Inspect → Toggle device toolbar (Ctrl+Shift+M)
3. Set custom size: 1024x500
4. Right-click page → Capture screenshot
5. Or use browser DevTools → More tools → Capture screenshot

### Option 3: Use Figma
1. Create new 1024x500px frame
2. Import the SVG logo
3. Add "PathQuest" text (Fraunces font, 72px, bold, white)
4. Add tagline "Track Your Summit Adventures" (24px, #b9a484)
5. Export as PNG

## Design Specs

- **Size**: 1024 x 500 pixels
- **Background**: Dark gradient (#1a1a1a to #2a2a2a)
- **Logo**: 192x192px, green gradient (#4A7C59 to #5A9C69)
- **Title**: "PathQuest" - 72px, Fraunces font, bold, white
- **Tagline**: "Track Your Summit Adventures" - 24px, #b9a484
- **Pattern**: Subtle topo pattern overlay

## Google Play Requirements

- ✅ Size: 1024 x 500 pixels
- ✅ Format: PNG or JPEG
- ✅ File size: Max 1MB
- ✅ No text overlay (text must be part of image)
- ✅ No promotional text (no "Free", "New", etc.)

## Notes

- The HTML version uses the Fraunces font (same as your app)
- The SVG version uses system fonts (may look slightly different)
- Both are ready to convert - choose whichever method is easiest for you!
